# <span id="HAPGEN">HAPGEN</span>

### \*.haps--input file & output file

|column| descrption|
|:----  |:----  |
|第1列|  每个 SNP 一行，每个单倍型一列，0、1表示等位基因编码|

### \*.leg--input file & output file

|column| descrption|
|:----  |:----  |
|第1列| SNP标识符，用于在不同研究之间对位点进行比较|
|第2列| 每个 SNP 的碱基对位置，通常以基对（bp）为单位表示|
|第3列| 等位基因1，A、G、C、T表示|
|第4列| 等位基因2，A、G、C、T表示|

### \*.legend--input file & output file

|column| descrption|
|:----  |:----  |
|第1列| SNP标识符，用于在不同研究之间对位点进行比较|
|第2列| 每个 SNP 的碱基对位置，通常以基对（bp）为单位表示|
|第3列| 等位基因1，A、G、C、T表示,通常是次要的|
|第4列| 等位基因2，A、G、C、T表示,通常是主要的|

### \*.map--input file
|column| descrption|
|:----  |:----  |
|第1列| 染色体位置，不受遗传连锁关系的影响|
|第2列| 标记右侧的以 cM/Mb 为单位的速率|
|第3列| 标记左侧的以 cM 为单位的累积速率|


### \*.gen--output file
|column| descrption|
|:----  |:----  |
|第1列| SNP位点标识符，标识作用|
|第2列| rsID，主要变体ID|
|第3列| 碱基对坐标，SNP位点在基因组上的位置|
|第4列| 等位基因，以A、G、C、T进行等位基因1、2的标识|
|第5列| 等位基因，以A、G、C、T进行等位基因1、2的标识|
|第6列| 单个SNP位点的等位基因情况在N个样本中的信息|
|第n列|同第6列|


### \*.sample--output file
|column| descrption|
|:----  |:----  |
|第1列| 家庭ID，通常是一个数字或字母编号，用于标识家庭|
|第2列| 家庭内ID，通常也是一个数字或字母编号，用于标识每个个体|
|第3列| 该样本是否缺失数据的标志，通常用0或1表示，0表示数据完整，1表示数据缺失|
|第4列| 性别代码（'1' = 男性，'2' = 女性，'0' = 未知）|
|第5列| 表型，'0' = 对照，'1' = 病例）或连续表型|


### \*.tags--output file
|column| descrption|
|:----  |:----  |
|第1列|  SNP 子集文件，标记的物理位置，与图例文件中的相匹配。|

# <span id="PLINK">PLINK</span>
### \*.QC--input file
|column| descrption|
|:----  |:----  |
|第1列| CHR，该位点所在的染色体编号|
|第2列| BP,碱基对坐标|
|第3列| SNP,单核苷酸多态性（SNP）标识符|
|第4列| A1,主要等位基因（major allele)|
|第5列| A2,次要等位基因（minor allele）|
|第6列| N，该位点的样本数|
|第7列| SE,该位点的标准误差（standard error）|
|第8列| P,评估该位点与研究特征之间的关联程度|
|第9列| OR,用于评估该位点与研究特征之间的关联程度|
|第10列| INFO,该位点的信息度，用于评估该位点的质量和可靠性，正相关|
|第11列| MAF,次要等位基因在样本中的出现频率。|
### \*.bed--input file

|column| descrption|
|:----  |:----  |
|第1列| 位点所在的染色体编号|
|第2列| 染色体或支架中特征的起始位置,染色体中的第一个碱基编号为0|
|第3列| 染色体或支架中特征的结束位置|





### \*.map--input file
|column| descrption|
|:----  |:----  |
|第1列| 染色体位置，这一列表示每个位点在基因组上的物理位置，通常以单位为bp（碱基对）表示。物理位置是基于基因组序列的，因此它不受遗传连锁关系的影响|
|第2列| 标记右侧的以 cM/Mb 为单位的速率|
|第3列| 标记左侧的以 cM 为单位的累积速率|

### \*.bim--input file
|column| descrption|
|:----  |:----  |
|第1列| 染色体编号，可以是任意字符串|
|第2列| SNP标识符，可以是任意字符串|
|第3列| 遗传距离，单位为摩根，一般为0|
|第4列| 物理位置，单位为bp（碱基对），一般为实际的基因组物理位置|
|第5列| 等位基因1的名称，一般为字母或数字，通常为次要|
|第6列| 等位基因2的名称，一般为字母或数字，通常为主要|

### \*.fam--input file
|column| descrption|
|:----  |:----  |
|第1列| 家系ID，可以是任意字符串|
|第2列| 个体ID，可以是除‘0’外的任意字符串|
|第3列| 父亲ID，如果没有父亲数据，则为0|
|第4列| 母亲ID，如果没有母亲数据，则为0|
|第5列| 性别，1表示男性，2表示女性，0表示未知|
|第6列| 出生日期，格式为YYYYMMDD，如果未知，则为0|


### \*.clumped--output file

|column| descrption|
|:----  |:----  |
|第1列| CHR,该位点所在的染色体编号|
|第2列| F，该位点与最显著信号的关联程度，即该位点的LD值|
|第3列| SNP,该位点的单核苷酸多态性（SNP）标识符|
|第4列| BP，碱基对坐标|
|第5列| P，该位点的P值，用于评估该位点与研究特征之间的关联程度|
|第6列| TOTAL，在该位点的LD窗口内共有多少个位点|
|第7列| NSIG，p ≥ 0.05 的聚集变体数|
|第8列| S05，聚类变体的数量，其中 0.01 ≤ p < 0.05|
|第9列| S01，聚类变体的数量，其中 0.001 ≤ p < 0.01|
|第10列| S001，聚类变体的数量，其中 0.0001 ≤ p < 0.001|
|第11列| S0001，聚类变体的数量，P值低于0.0001|
|第12列| SP2，p < --clump-p2 阈值的成员的逗号分隔 ID 和文件编号|

# <span id="IMPUTE2">IMPUTE2</span>
### \*.vcf--input file
|column| descrption|
|:----  |:----  |
|第1列| 染色体编号，例如 "1"、"2"、"X" 等|
|第2列| 位点位置，即在染色体上的位置|
|第3列| 参考等位基因，即在参考基因组中所定义的该位点的碱基|
|第4列| 可变等位基因，即在该位点上存在的其他可能的碱基|
|第5列| 质量分数，表示该位点的可靠程度|
|第6列| 过滤标记，表示该位点是否通过了质量控制|
|第7列| 包含了一系列的注释信息，例如变异类型、功能影响、频率等|
|第8列| 格式标记，表示后续的样本数据的格式。|
|第9列| 样本数据列，包含了每个样本在该位点上的基因型信息，例如 "0/1" 表示该样本的两个等位基因分别为参考等位基因和可变等位基因。这些样本数据列的列名即为样本的编号。|
|第n列| 同第9列|

### \*.assoc--output file

SNP CHR BP A1 A2 P OR
|column| descrption|
|:----  |:----  |
|第1列| SNP位点的标识符|
|第2列| 该位点所在的染色体编号|
|第3列| 碱基对坐标|
|第4列| 该位点的第一等位基因（通常次要）|
|第5列| 该位点的第二等位基因（通常主要）|
|第6列| 该位点的 P 值，表示该位点与表型数据之间的关联程度|
|第7列| 几率（等位基因 1 | 病例）/几率（等位基因 1 | 对照）|
### \*.pheno--output file

|第2列| 个体 ID，表示样本的唯一标识符|
|第3列| 表型数据，通常是身高、体重、疾病等|


### \*.best--output file
|column| descrption|
|:----  |:----  |
|第1列| 家系 ID，表示样本所属的家庭或群体|
|第2列| 个体 ID，表示样本的唯一标识符|
|第3列| 是否被包括在回归分析。1已被纳入回归分析；0未被纳入回归分析|
|第4列| 个体多基因风险评分|
